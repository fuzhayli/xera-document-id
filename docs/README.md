# XERA Document ID System Docs

Bu klasor, offline HTML generator'dan cok kullanicili Document ID Request
and Approval System'e gecis icin hazirlanan calisma dokumanlarini icerir.

## Dokumanlar

| Dosya | Icerik |
|---|---|
| `document-id-rule-matrix.md` | PPTX ve Excel gorunumlerinden cikarilan document ID, filename, approval ve validation kurallari. |
| `frontend-prototype-assessment.md` | Mevcut HTML prototipin hangi kisimlarinin korunacagi, hangi mantigin backend'e tasinacagi ve MVP ekran/veri modeli taslagi. |
| `backend-api.md` | Node + SQLite backend iskeleti, endpointler, seed kullanicilar ve sequence karari. |

## Siradaki Adim

Mevcut HTML prototipi backend API'ye baglanarak request formu ve admin
approval ekrani uygulamaya alinabilir.
